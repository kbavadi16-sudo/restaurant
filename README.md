# Kwality Restaurant

## Project Documentation

### Overview
Kwality Restaurant is a modern web application designed to provide a seamless experience for customers looking to browse our menu, make reservations, and learn more about our services. Our website features an intuitive design, ensuring that users can easily navigate and find the information they need.

### Features
- **Online Menu**: View our extensive menu with detailed descriptions of each dish.
- **Reservation System**: Make reservations online with ease.
- **Location and Hours**: Find our restaurant's location and hours of operation.
- **Contact Us**: A section for inquiries and feedback.

### Technologies Used
- **HTML/CSS**: For structure and styling of the website.
- **JavaScript**: For interactive elements and functionality.
- **Node.js**: Backend application logic.
- **MongoDB**: Database for storing user reservations and menu items.

### Setup Instructions
1. Clone the repository: `git clone https://github.com/kbavadi16-sudo/restaurant`
2. Navigate to the project directory: `cd restaurant`
3. Install dependencies: `npm install`
4. Start the server: `npm start`
